nesR
====

Code to scrape data from the National Eutrophication Survey archival PDF

Installation
------------

You can install nesR from github with:

``` r
# install.packages("devtools")
devtools::install_github("jsta/nesR")
```

Usage
-----

``` r

parse_nes(tif_file)
```
